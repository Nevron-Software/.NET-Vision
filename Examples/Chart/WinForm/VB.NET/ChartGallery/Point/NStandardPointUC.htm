This example demonstrates a standard point chart.
You can control the shape of the points from the Point Style Combo.
The Point Size scroll controls the size of the points.
When you check the Inflate Margins check the axis scales are adjusted to fit the entire point series within the chart area.
The Left Axis Round To Tick check controls whether the left axis scale min and max values should be adjusted to end on exact tick marks.
If you check the Different Colors check box the points will be displayed in different colors. If this check box is not checked you can control the filling of the points from the Point Fill Style button.
