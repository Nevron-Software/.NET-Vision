This example demonstrates stacked bar chart. The first series MultiBarMode is set to Series, while the second and third bar series are displayed with MultiBarMode set to Stacked or StackedPercent.
Two types of stacking are supported - Stacked and Stacked Percent. The first type simply piles the bars on top of each other. The stacked percent style displays the contribution of each individual stack to the overall pile sum.
Stack Style combo - changes the stacking style
The Positive Data button changes the data with random positive values. The Positive and Negative Data button changes the data with random positive and negative values. Note that in Stacked percent mode all data is processed by its absolute value.
Data Labels combos control the information displayed on each stack.
