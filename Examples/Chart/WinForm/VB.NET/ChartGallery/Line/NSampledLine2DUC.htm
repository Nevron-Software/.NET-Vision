This example demonstrates how to enable sampling for the line series.
When using sampling the line will ignore settings for markers and data labels and will also automatically resample the data storage depending on the size of the chart on the screen. This allows for the visualization of massive amounts of data.
The sample distance scroll bar on the right allows you to modify the sampling distance
Click on the "Add 20, 000 Data Points" or "Add 40, 000 Data Points" to add some more new random data.
The data point counter below shows the current number of data points in thousands.
Press the "Clear Data" button to clear all data points and start over.
