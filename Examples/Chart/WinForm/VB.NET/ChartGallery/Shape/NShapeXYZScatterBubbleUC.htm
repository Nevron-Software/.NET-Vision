This example demonstrates how to use the Shape Series to display a XYZ scatter bubble chart.
You can change the fill styles applied to the bubbles by pressing the Bubble Fill Style buttons.
The Inflate Margins check fits the bubbles in the Chart Area.
The Axes Round To Tick check makes the PrimaryY, PrimaryX and Depth axes begin and end on exact tick marks.
The shape of the bubbles is controlled by the Bubble Shape combo.
The Change X Values button changes the X positions of the bubbles.
The Change Z Values button changes the Z positions of the bubbles.
The Change Y Values button changes the Y positions of the bubbles.
