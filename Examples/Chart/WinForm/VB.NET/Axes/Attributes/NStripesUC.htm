Stripes are one of the most useful objects when it comes to highlighting information in the chart area. They allow you to highlight a range of axis values.

Begin Value scroller - controls the beginning of the stripe
End Value scroller - controls the ending of the stripe
Include in axis range - whether the begin / end values of the stripe will be included in the axis range Stripe Fill Style button - displays the stripe Fill Style editor
Show At Wall checks - control whether the respective stripe should be displayed on the specified chart wall.
