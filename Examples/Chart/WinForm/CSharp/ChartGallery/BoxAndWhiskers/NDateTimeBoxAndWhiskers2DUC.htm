Box and whiskers plots are very helpful in interpreting the distribution of data. Each box and whisker item represents a set of values and displays statistical information for it like minimum, maximum and median values, upper and lower quartiles, outliers and optionally a mean value.
This example demonstrates a box and whiskers series with custom date time values. This chart lets you compare the statistical parameters of several data sets that are produced in different moments in time.
You can use the controls to modify some properties of the box and whiskers series like box size, whiskers size, fill styles, stroke styles, etc.
