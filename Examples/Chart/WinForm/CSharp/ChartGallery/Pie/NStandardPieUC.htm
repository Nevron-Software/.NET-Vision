This example demonstrates a 3D Pie Chart.
You can select the shape of the pie sectors from the Pie Shape combo.
For the different shapes you can control several parameters like edge size, outer radius and inner radius.
You can select the display mode of the data point labels from the "Pie Label Mode" combo.
The Begin Angle scroller controls the begin angle of the first pie slice.
The Total Angle scroller controls the total angle of the pie.
