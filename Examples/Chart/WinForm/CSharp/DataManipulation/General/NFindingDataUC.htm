The data series of the component have built-in finding functions.
In the case when the data series contains objects of type double you can perform the following finding operations:
- find the index of the min value
- find the index of the max value
- find the index of the first item in the series with the specified value

In the case when the data series contains objects of type string you can find the index of the first string that is equal to an user specified string.
