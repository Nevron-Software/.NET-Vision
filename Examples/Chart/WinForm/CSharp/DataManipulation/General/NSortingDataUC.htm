The component has build data sorting capabilities. Sorting can be applied to individual data series as well as on collections of data series. In the second case you can specify a master data series on which the sorting is performed.
Change Data button - changes the Values, Labels and FillStyles data series of the bar series.
Sort Order combo - specifies the sort order in which the data series should be sorted
Sort Values Only - sorts only the Values data series of the bar.
Sort All On Values - sorts the Values, Labels and FillStyles data series using the Values data series as master series
Sort All On Labels - sorts the Values, Labels and FillStyles data series using the Labels data series as master series
