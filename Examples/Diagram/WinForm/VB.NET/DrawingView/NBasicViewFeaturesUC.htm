This example demonstrates the basic drawing view features, which are:

Panning. Panning is the the ability to control the viewport origin. The viewport origin is the first point in document scene coordinates, which is displayed by the view. The viewport origin is mapped to the window origin, which is the first point in view client coordinates which displays document content.

Global visibility gives you control over the global visibility of particular elements, decorations and effects.

Document padding lets you apply a padding to the document bounds. The total viewport size is calculated by inflating the document bounds with the document padding.

Document shadow. The view provides you with the ability to display a decoration shadow behind the document bounds.

Scaling and zooming. The view provides you with a natural way to apply X and Y scaling of the document. In case the X and Y scales are equal the effect of scaling is called zooming.

View layout. The component support for different types of view layouts:

  • Normal - the user can freely modify the viewport origin and scale parameters of the view (pan and zoom the view)
  • Fit - the viewport origin and scales are automatically calculated, so that the document fits in the view window with preserved aspect ratio.
  • Stretch - the viewport origin and scales are automatically calculated, so that the document stretches to fill the window. This setting can break the aspect ratio of the document (scale it irregularly).
  • Stretch to Width - the viewport origin X and scales are automatically calculated, so that the document stretches to fit the window width. This setting preserves the document aspect ratio.
  • Stretch to Height - the viewport origin Y and scales are automatically calculated, so that the document stretches to fit the window height. This setting preserves the document aspect ratio.

Scrolling and autoscrolling. The view exposes control over different aspects of scrolling and autoscrolling, such as:

  • Scrollbars visibility
  • Alignment of the document relative to the view, when no scrollbars should be displayed
  • Small scroll step (defined in pixels)
  • Delay for autoscrolling (specified in milliseconds)
  • Time between two autoscroll steps (specified in milliseconds)
