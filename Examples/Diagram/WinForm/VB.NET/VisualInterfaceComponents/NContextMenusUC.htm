This example demonstrates how to implement custom commands and show them in the context menus.

In this example we have created a custom diagram command, which is represented by the yellow smile face (perform a right click in the designer to show a context menu - the command should be last in the list). It will show a message box when clicked.

In order to display a custom command in the context menus you have to do the following:

  1. Register the commmand in the diagram commander commands collection
  2. Override the context menus builder in order to place the command in context menus
  3. Recreate the command bars
