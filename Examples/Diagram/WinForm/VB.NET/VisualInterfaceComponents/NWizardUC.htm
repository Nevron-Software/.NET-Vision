This example demonstrates the built in support for programmable templates as well as the wizard, which can be used for their visual selection and editing.

In this example we have created a custom template named "My template", which is pluged in the wizard.

To show the wizard - click on the Show Wizard button in the example form. Note that there is a category called "My Templates". The one and only template in this category is the "My template" template, the source code of which is provided with this example.
