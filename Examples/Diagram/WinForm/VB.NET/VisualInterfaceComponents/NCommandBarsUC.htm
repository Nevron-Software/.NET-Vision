This example demonstrates how to implement custom commands and place them on the command bars.

In this example we have created a custom diagram command, which is represented by the yellow smile face. It will show a message box when clicked.

In order to display a custom command in the toolbars you have to do the following:

  1. Register the commmand in the diagram commander commands collection
  2. Override the toolbars builder in order to place the command in a specific toolbar
  3. Recreate the command bars
