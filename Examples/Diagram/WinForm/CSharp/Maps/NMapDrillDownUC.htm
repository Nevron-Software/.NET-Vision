This example demonstrates the map drip down capabilities of Nevron Maps. Click a state to see a map of its counties. Then click a county to see its FIPS code or click outside the state (in the blue area) to go back to the States map.

When you click on a state the code the NodeMouseDown event of the drawing document's event sink service is fired and the code in it loads the map of the clicked state in the place of the USA map. Then when you click a county the code changes its style sheet and text.
