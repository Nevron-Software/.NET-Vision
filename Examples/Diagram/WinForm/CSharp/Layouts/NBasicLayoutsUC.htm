This example demonstrates the built-in support for basic layouts, which is implemented by the NBatchLayout class.

The NEntityLayout is used to wrap the most commonly used operations for layout, which are a must for every drawing application. These are:

  1. Align operations
  2. Resize operations
  3. Spacing operations
  4. Center in document operations

To experiment with their behavior just select the nodes to which you want to apply some of these operations and click on the respective button in the Example form.

It is also worth mentioning that most of the operations take into account the current selection anchor. For example if you select several nodes and click on the Align - Lefts button, the nodes will be aligned to the left side of the anchor node. The anchor node is easily distinguished, because its handles are displayed with inverted colors.

Note that the operations record history transactions - after you perform some layout operation, just expand the Undo list (located in the Standard Commands toolbar) to see the transaction that was recorded.

All layout operations take into account the node resize and move permissions.
