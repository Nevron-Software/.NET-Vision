This example demonstrates the built in support for AutoCAD Drawing Interchange (DXF) export, which is implemented by the NAutocadExporter extension. When you click on the <Export to DXF> button a DXF file is generated and saved in the application's startup directory. The file is then opened with the default viewer for DXF files (e.g. AutoCAD, DWG TrueView, etc).

Exporting to AutoCAD is as easy as writing 2 lines of code:

NAutocadExporter exporter = new NAutocadExporter(document);
exporter.SaveToFile(fileName);
