This example demonstrates the built in support for XAML export, which is implemented by the NXamlExporter extension. When you click on the <Export to XAML> button a XAML file is generated and saved in the application's startup directory. The file is then opened with your default XAML viewer.

Exporting to Flash is as easy as writing 2 lines of code:

NXamlExporter exporter = new NXamlExporter(document);
exporter.SaveToFile(fileName);
