This example demonstrates how to display a simple state diagram. It uses the following features:

  • Predefined Flow Charting Shapes - the component comes with built in support for a myriad of commonly used flowcharting shapes.
  • 1D and 2D Shapes - shows how to create 1D and 2D shapes and how to connect the plugs of 1D shapes.
  • Connectors - demonstates the different types of predefined connectors.
  • Style sheets - demonstates how to assign stylesheets to shapes. In this examples all 1D shapes inherit styles from the connectors stylesheet.
