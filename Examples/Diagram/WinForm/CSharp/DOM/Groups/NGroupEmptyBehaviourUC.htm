This example demonstrates how to instruct groups to automatically destroy themselves, when they become empty and how to prevent groups from becoming empty.

To explore how autodestruction works perform the following steps:

  1. Select the group
  2. In the Example form check the Auto destroy if empty check box
  3. Drill down to select a group child shape
  4. Press the Del button
  5. Repeat steps 3 and 4 until all group child shapes are deleted

At the end of this operation the group will automatically be destroyed. If you did not check Auto destroy if empty check the group will still exist, but it wont have any child nodes.

To explore how empty prevention works perform the following steps:

  1. Reset the example
  2. Select the group
  3. In the Example form uncheck the Can be empty check box
  4. Drill down to select a group child shape
  5. Press the Del button
  6. Repeat steps 4 and 5 until all group child shapes are deleted

When you try to delete the last shape an error message will pop up to indicate that this operation is not allowed.
