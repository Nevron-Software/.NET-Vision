This example demonstrates shape ports.

Ports are one of the optional shape elements (the other two optional elements are control points and labels). Ports are connection points to which multiple plugs can be attached. Ports can be anchored to a specific aspect of the shape to which they belong, or to any model inside this shape. The actual model to which a port is anchored is called anchor model.

Currently Nevron Diagram for .NET supports the following types of ports:

  • Dynamic port - this is a port, whose location is aligned relative to the rotated bounds of the anchor model. Unlike the rest of the ports, which glue the connected plugs to the port location, the dynamic port can be configured to glue the plugs in several modes.
  • Bounds port - this is a port, whose location is aligined relative to the scene bounds of the anchor model.
  • Rotated bounds port - this is a port, whose location is aligined relative to the scene protection of the model bounds of the anchor model.
  • Logical line port - this is a port, whose location is positioned at the percent position along the logical line of the acnhor model.

To explore the ports behavior just select a shape and modify the port properties which will appear in the Example form.
