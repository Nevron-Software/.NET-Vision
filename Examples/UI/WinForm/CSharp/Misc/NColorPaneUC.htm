Nevron User Interface 4.0 introduces lots of color tools, one of which is a generic pane used to visualize 143 standard colors, found in the color picker of MS Office 2003. The pane is a predefined NCommandPane and all the cells are nothing but predefined NCommand objects. It has lots of customization features as well as control over size of a single cell.
  • Unified overall appearance entirely based on pluggable renderers and palettes.
  • User controllable cell size. By default equals (26, 26).
  • Selectable - specifies whether the pane can receive focus (either on mouse down or using the Tab key). False by default.
  • ShowTooltips - specifies whether a tip containing RGB and HTML representation of a color will be displayed when a cell is hovered.
  • ColorChangeStyle - controls the way of firing ColorChanged event. By default it is fired when the user releases the mouse.
  • Color property - gets or sets the color value of the pane. If the specified color does not exist in the collection Color.Empty is returned.
