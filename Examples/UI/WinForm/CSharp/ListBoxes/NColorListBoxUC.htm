NColorListBox is a predefined NListBox, used for displaying colors rather than images. Use this control when you need a color selection tool - simply add colors as items.
  • PopulateSystemColors method - fills the color listbox with all the color values specified in the SystemColors enumeration.
  • PopulateWebColors method - fills the color listbox with all the color values specified in the KnownColor enumeration which are not system.
  • PopulateKnownColors method - fills the color listbox with all the color values specified in the KnownColor enumeration.
  • Color property - tries to extract the color value from the selected item. If such is not found returns Color.Empty.
