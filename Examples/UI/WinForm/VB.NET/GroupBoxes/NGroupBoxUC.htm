Nevron User Interface provides provides you with custom drawn group boxes. Select among two predefined styles - standard and line at top only. Support for images from ImageList also. Note that when ColorScheme is set to WindowsDefault the groupbox will be painted as Windows paints it.
  • ImageList and ImageIndex - specify these and the GroupBox will now display image left before text.
  • Style - specify your preffered GroupBoxStyle - standard or line at top.
  • Text origin - user controllable origin of the text.
  • Native support for WindowsXP® visual styles.
