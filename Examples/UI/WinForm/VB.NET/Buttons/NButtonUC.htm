Nevron User Interface Suite 4.0 for .NET extends the standard WindowsForms Button control and gives it a stylish appearance that will satisfy your "hunger" for something fresh and modern.
  • Unified overall appearance entirely based on pluggable renderers and palettes.
  • Border support - specify your favourite border style and the base color used for border drawing.
  • User definable focus rect - specifies whether the button will display focused rectangle when selected.
  • Selectable - specifies whether the button can receive focus or not.
  • ImageSize - controls the size used to display the button's image.
